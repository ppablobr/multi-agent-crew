```
LinkedIn ←→ n8n (posts sobre workflows)
LinkedIn ←→ AI PM (campanha de conteúdo)
LinkedIn ←→ Process Manager (divulgar processos)
LinkedIn ←→ Education (divulgar programas educacionais)
LinkedIn ←→ Data Analytics (posts com insights e dados)
n8n ←→ Process Manager (automatizar processos)
n8n ←→ Education (criar workflows de exemplo para trilhas)
n8n ←→ Data Analytics (medir ROI de automações)
n8n ←→ DevOps (organizar workflow files e exports)
AI PM ←→ Process Manager (documentar projeto)
AI PM ←→ Agent Manager (criar agentes do projeto)
AI PM ←→ Education (integrar iniciativas educacionais ao roadmap)
AI PM ←→ Data Analytics (métricas de projetos de IA)
Education ←→ Designer (criar diagramas e visuais para materiais)
Education ←→ Data Analytics (avaliar programas educacionais)
Process Manager ←→ Data Analytics (KPIs de processos)
Process Manager ←→ DevOps (organizar process docs)
Process Manager ←→ Tech Docs Writer (documentar SOPs)
Designer ←→ Data Analytics (visualizações bonitas)
Designer ←→ DevOps (organizar design assets)
Tech Docs Writer ←→ DevOps (atualizar docs após reorg)
DevOps ←→ Agent Manager (estrutura de novos agentes)
Agent Manager ←→ todos (criar/evoluir qualquer agente)
```
