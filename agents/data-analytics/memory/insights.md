## Fórmulas e Cálculos Comuns

### ROI de Automação
```
Tempo economizado (horas/mês) = (Tempo manual por tarefa - Tempo automatizado) × Tarefas por mês
Economia de custo = Tempo economizado × Custo por hora
ROI % = (Economia total - Custo de setup) / Custo de setup × 100
Payback (meses) = Custo de setup / Economia mensal
```
